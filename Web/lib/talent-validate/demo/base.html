1. 先把dist目录下的资源和jquery.js(或zepto.js)拷贝到您的工程中。本框架对jquery的版本几乎没有特别要求(依赖很小)，然后在你的页面加引入以下资源 
    						<link type="text/css" rel="stylesheet" href="../js/validate/css/validate.css" />
						<script src="./jquery.js" language="javascript"></script>   //或zepto.js
						<script src="../js/validate/talent-validate-all.js" language="javascript"></script>
  2. 你的页面应该会有一些需要验证的元素譬如 
    						<input type='text' name='name_1' />
						<input type='text' name='name_2' />
						<input type='text' id='id_1' />
						<input type='text' id='id_2' />
  3. 绑定需要验证的元素以及验证规则 
    						tt.vf.req.add("name_1", "name_2");     //用name绑定需要验证的元素
						tt.vf.email.addId("id_1", "id_2");     //用id绑定需要验证的元素
  4. 在页面提交前对表单进行验证 
    						if(!tt.validateForm('loginForm')) {   //loginForm是form的name
							return false;
						}
  5. 本框架是为验证的灵活性而生的，它有丰富的API供开发人员使用，譬如只验证某个元素，根据元素值灵活解绑验证等。 
    						//请参考其它API和后面的各种例子
